# High Conviction Signals

This directory contains high conviction trading signals generated by `make stocks`.

## Structure

```
high_conviction/
├── buy/                    # Strong BUY signals (P(r>0) ≥ 62%, E[return] ≥ 2%)
│   ├── AAPL_1d.json       # Individual signal files
│   ├── AAPL_3d.json
│   └── manifest.json      # Summary of all buy signals
└── sell/                   # Strong SELL signals (P(r>0) ≤ 38%, E[return] ≤ -2%)
    ├── TSLA_7d.json
    └── manifest.json
```

## Signal File Format

Each signal JSON file contains:

```json
{
  "ticker": "AAPL",
  "asset_label": "Apple Inc. (AAPL)",
  "sector": "Information Technology",
  "horizon_days": 3,
  "signal_type": "STRONG_BUY",
  "probability_up": 0.7234,
  "probability_down": 0.2766,
  "expected_return_pct": 3.45,
  "expected_profit_pln": 34500.00,
  "signal_strength": 0.3234,
  "conviction_probability": 0.7234,
  "generated_at": "2026-02-05T10:30:00",
  "price_history": {
    "prices": [...],
    "current_price": 185.50,
    "price_change_pct": 2.3
  },
  "options_chain": {
    "expiration_dates": ["2026-02-21", "2026-02-28"],
    "options": [...],
    "atm_options": [...]
  }
}
```

## Options Data

- **BUY signals**: Include CALL options data
- **SELL signals**: Include PUT options data

Options are filtered to:
- Expirations within 30 days
- Strikes within 20% of current price
- Maximum 50 contracts per signal

## Regeneration

Signals are **completely regenerated** each time `make stocks` runs.
Previous versions are deleted before new signals are written.

## Thresholds

| Signal Type | Probability | Expected Move |
|-------------|-------------|---------------|
| STRONG_BUY  | P(r>0) ≥ 62% | E[return] ≥ +2% |
| STRONG_SELL | P(r>0) ≤ 38% | E[return] ≤ -2% |
